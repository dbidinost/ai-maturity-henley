3. Upload your file

In your new repo, click Add file → Upload files
Drag your ai_maturity_final.html onto the page
Rename it to index.html before uploading, or GitHub Pages won't serve it as the default page
Click Commit changes

4. Enable GitHub Pages

Go to your repo Settings → scroll down to Pages in the left sidebar
Under Source, select Deploy from a branch
Branch: main, folder: / (root)
Click Save

5. You're live
After about a minute your site is at:
https://yourusername.github.io/ai-maturity-henley
That's your QR code URL.
